# ALQari — Arabic-first Document Intelligence APIs

ALQari helps developers convert Arabic and English documents into structured data using OCR, extraction, validation, and document intelligence APIs.

## What you can build

- Arabic OCR for printed and handwritten documents
- Document data extraction
- KYC and onboarding automation
- Invoice and form processing
- Legal and government document processing
- Search and chat over documents
- Human validation workflows

## Quick Start

1. Create a free developer account
2. Generate your API key
3. Upload a document
4. Run OCR or extraction
5. Receive structured JSON output
